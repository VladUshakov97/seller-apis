# Скрипт синхронизации остатков и цен с Ozon
### Назначение
Скрипт автоматически обновляет информацию о ценах и остатках товаров в интернет-магазине Ozon, используя данные с сайта `timeworld.ru`.

### Что делает скрипт
1. Загружает актуальный файл с остатками товаров с внешнего сайта (Casio).
2. Получает список товаров из вашего магазина на Ozon.
3. Сравнивает внешние данные с тем, что уже загружено в Ozon.
4. Обновляет остатки на складе Ozon
5. Обновляет цены на Ozon в соответствии с файлом (конвертируя цену из формата `5'990.00 руб.` в `5990`).
6. Удаляет временный файл после обработки.
